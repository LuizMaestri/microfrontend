# microfrontend
Application using single-SPA for testing microfrontend

## Running Application

### Running React Parcel
Run on port 8080
```sh
cd packages/counter
yarn start
```

### Running Vue Parcel
Run on port 8081
```sh
cd packages/todo-list
yarn serve
```

### Running Single SPA
Run on port 9000
```sh
yarn start
```
